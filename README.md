# 🚗 Dashboard Power BI - Locadora de Carros

Projeto desenvolvido no Power BI com foco na análise de dados de uma locadora de carros, permitindo acompanhar indicadores importantes do negócio de forma visual e estratégica.

---

## 📊 Objetivos do Projeto

O dashboard foi criado para facilitar a visualização e análise de informações relacionadas à operação da locadora, auxiliando na tomada de decisão.

Principais análises:
- Receita total
- Quantidade de aluguéis
- Veículos mais alugados
- Clientes com maior volume de locações
- Desempenho por período
- Indicadores gerais da empresa

---

## 🛠️ Ferramentas Utilizadas

- Power BI
- Power Query
- DAX
- Modelagem de Dados
